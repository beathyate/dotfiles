# My Dotfiles

To install:

```bash
$ rake
```

This will symlink all files and directories in `./source/*` to `~/.FILENAME`

## Requirements

* Xcode.app + Command Line Tools
* Posgres.app
* [homebrew](http://mxcl.github.io/homebrew/)
* `rbenv` + `ruby-build` + `bundler`
* `vcprompt`
* Heroku Toolbelt
* TextMate 2
